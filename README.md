# HR System

This project is a Smart HR System aimed at managing employees, roles, and attendance efficiently.

## Project Idea
- Manage employee data (add, edit, delete, view)
- Manage roles and departments
- Track attendance
- Basic reports and statistics

## Technologies
- ASP.NET Core
- Entity Framework Core
- SQL Server (or any other database)
- HTML/CSS/JavaScript for frontend (optional at this stage)

## Note
The project is still in the planning phase and will be updated continuously during development.
